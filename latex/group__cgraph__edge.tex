\doxysubsubsubsubsection{Подробное описание}
An abstract edge has two endpoint nodes called tail and head where all outedges of the same node have it as the tail value and similarly all inedges have it as the head. In an undirected graph, head and tail are interchangeable. If a graph has multi-\/edges between the same pair of nodes, the edge\textquotesingle{}s string name behaves as a secondary key.

Note that an abstract edge has two distinct concrete representations:\+ as an in-\/edge and as an out-\/edge. In particular, the pointer as an out-\/edge is different from the pointer as an in-\/edge. The function \doxylink{group__cgraph__edge_ga8b777558e7d3375ab4e6cfd4f22c4528}{ageqedge} canonicalizes the pointers before doing a comparison and so can be used to test edge equality. The sense of an edge can be flipped using \doxylink{group__cgraph__edge_ga21ee14460f49ddb2e954f3cf9f9100a3}{agopp}.
