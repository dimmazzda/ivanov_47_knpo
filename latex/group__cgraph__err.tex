\doxysubsubsubsubsection{Подробное описание}
The library provides a variety of mechanisms to control the reporting of errors and warnings. A message is only written if its type has higher priority than a programmer-\/controlled minimum, which is \doxylink{group__cgraph__err_gga0c75ffdbe1d93ca74fe77dc46a45a6cfa4486cfed6fa2babf34e58bdb3707461c}{AGWARN} by default. The programmer can set this value using \doxylink{group__cgraph__err_gac8b80f854859b4017499e42067bae5b7}{agseterr}, which returns the previous value. Calling {\ttfamily agseterr(\+AGMAX)} turns off the writing of messages.

The function \doxylink{group__cgraph__err_ga42ef64ab2933b0503ccc8406dfe375b7}{agerr} is the main entry point for reporting an anomaly. The first argument indicates the type of message. Usually, the first argument is \doxylink{group__cgraph__err_gga0c75ffdbe1d93ca74fe77dc46a45a6cfa4486cfed6fa2babf34e58bdb3707461c}{AGWARN} or \doxylink{group__cgraph__err_gga0c75ffdbe1d93ca74fe77dc46a45a6cfa76eb3bf18a02c1e78b1be01b25c296f2}{AGERR} to indicate warnings and errors, respectively. Sometimes additional context information is only available in functions calling the function where the error is actually caught. In this case, the calling function can indicate that it is continuing the current error by using \doxylink{group__cgraph__err_gga0c75ffdbe1d93ca74fe77dc46a45a6cfabd794ca4c498f8e4c0cf4d26755e0398}{AGPREV} as the first argument. The remaining arguments to \doxylink{group__cgraph__err_ga42ef64ab2933b0503ccc8406dfe375b7}{agerr} are the same as the arguments to {\ttfamily printf}.

The functions \doxylink{group__cgraph__err_ga1b3c7fe6147832a992f62fcd28e79a82}{agwarningf} and \doxylink{group__cgraph__err_gaf80e0604e4bcff00064559e46261b8d4}{agerrorf} are shorthand for {\ttfamily agerr(\+AGWARN,...)} and {\ttfamily agerr(\+AGERR,...)}, respectively.

Some applications desire to directly control the writing of messages. Such an application can use the function \doxylink{group__cgraph__err_gadee66a17e220d081d595875a1afc0c2c}{agseterrf} to register the function that the library should call to actually write the message. The previous error function is returned. By default, the message is written to {\ttfamily stderr}.

Errors not written are stored in a log file. The last recorded error can be retrieved by calling \doxylink{group__cgraph__err_ga4776647f453831f116fee7e200ee2823}{aglasterr}. Unless the printing of error messages has been completely disabled by a call to {\ttfamily agseterr(\+AGMAX)}, standard error must not be wide-\/oriented, even if a user-\/provided error printing function is provided.

The function \doxylink{group__cgraph__err_gac3dd9f429c7f6f0b7147e8ffd5c83ed5}{agerrors} returns non-\/zero if errors have been reported.
