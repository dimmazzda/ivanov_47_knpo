\doxysubsection{Подробное описание}
{\bfseries{libxdot}} provides support for parsing and deparsing graphical operations specified by the xdot language. \href{https://graphviz.org/docs/outputs/canon/\#xdot}{\texttt{xdot}} is extended dot format containing complete layout information.
