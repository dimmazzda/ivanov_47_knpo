\doxysubsubsubsection{Подробное описание}
Programmer-\/defined values may be dynamically attached to graphs, subgraphs, nodes, and edges. Such values are either character string data (see \doxylink{group__cgraph__attr_ga7523d2c3b09fce2d069cc8f98f6b156f}{agattr\+\_\+text}, \doxylink{group__cgraph__attr_ga7550a6132763746371a013a614d23a28}{agattr\+\_\+html}, and \doxylink{group__cgraph__attr_gac9976dbfd1f6494265db70a8b5757e13}{agattr}) (for I/\+O) or uninterpreted binary \doxylink{group__cgraph__rec}{records} (for implementing algorithms efficiently).

{\itshape String attributes} are handled automatically in reading and writing graph files. A string attribute is identified by name and by an internal symbol table entry (\doxylink{group__cgraph__rec_ga51d4ce67ee895cc7051b19db92d771d6}{Agsym\+\_\+t}) created by Libcgraph. Attributes of nodes, edges, and graphs (with their subgraphs) have separate namespaces. The contents of an \doxylink{group__cgraph__rec_ga51d4ce67ee895cc7051b19db92d771d6}{Agsym\+\_\+t} have a char\texorpdfstring{$\ast$}{*} {\itshape name} for the attribute\textquotesingle{}s name, a char\texorpdfstring{$\ast$}{*} {\itshape defval} field for the attribute\textquotesingle{}s default value, and an int {\itshape id} field containing the index of the attribute\textquotesingle{}s specific value for an object in the object\textquotesingle{}s array of attribute values.
