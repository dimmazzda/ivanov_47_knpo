\doxysubsection{Подробное описание}
{\bfseries{Libcgraph}} supports graph programming by maintaining graphs in memory and reading and writing graph files. Graphs are composed of nodes, edges, and nested subgraphs. These graph objects may be attributed with string name-\/value pairs and programmer-\/defined records (see Attributes). Most of Libcgraph’s global symbols have the prefix {\bfseries{ag}} (case varying). In the following, if a function has a parameter {\ttfamily int createflag} and the object does not exist, the function will create the specified object if {\ttfamily createflag} is non-\/zero; otherwise, it will return NULL.
