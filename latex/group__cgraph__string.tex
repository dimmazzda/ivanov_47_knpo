\doxysubsubsubsubsection{Подробное описание}
Storage management of strings as reference-\/counted strings. The caller does not need to dynamically allocate storage.

All uses of cgraph strings need to be freed using \doxylink{group__cgraph__string_gaa0bb43286b278c334a2db74b0d699338}{agstrfree} in order to correctly maintain the reference count.
