\doxysubsubsubsubsection{Подробное описание}
By default, nodes are stored in ordered sets for efficient random access to insert, find, and delete nodes.

\doxylink{group__card_ga4adc84a502d28ec87c931b98407846fd}{agnnodes}, \doxylink{group__card_gab6fefbed26b04eadf563f407a7d3774a}{agnedges}, and \doxylink{group__card_ga2995244a8e345f4574876251e8b449d2}{agnsubg} return the sizes of node, edge and subgraph sets of a graph.

The function \doxylink{group__card_ga7bde08e6e67433a4afd6c046e0b62d9b}{agdegree} returns the size of a node’s edge set, and takes flags to select in-\/edges, out-\/edges, or both.

The function \doxylink{group__card_gab2d5ab6810e74b55cf287bcb8490a8be}{agcountuniqedges} returns the size of a node’s edge set, and takes flags to select in-\/edges, out-\/edges, or both. Unlike \doxylink{group__card_ga7bde08e6e67433a4afd6c046e0b62d9b}{agdegree}, each loop is only counted once.
