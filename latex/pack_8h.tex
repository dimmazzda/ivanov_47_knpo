\doxysubsection{Подробное описание}
{\bfseries{libpack}} supports the use of connected components in the context of laying out graphs using other graphviz libraries. One set of functions can be used to take a single graph and break it apart into connected components. A complementary set of functions takes a collection of graphs (not necessarily components of a single graph) which have been laid out separately, and packs them together.

As this library is meant to be used with {\ttfamily libcommon}, it relies on the \doxylink{struct_agraphinfo__t}{Agraphinfo\+\_\+t}, \doxylink{struct_agnodeinfo__t}{Agnodeinfo\+\_\+t} and \doxylink{struct_agedgeinfo__t}{Agedgeinfo\+\_\+t} used in that library.
